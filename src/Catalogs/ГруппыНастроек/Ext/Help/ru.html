Для каждого пользователя могут быть сделаны персональные настройки, применяемые при его работе в системе.

Справочник «Группы настроек» содержит наименования групп настроек, используемых в отчетах. Каждая группа настроек определяет индивидуально настроенный вид отчетов при  работе с конфигурацией пользователя, у которого есть права на эту группу настроек.

Из справочника  «Группы настроек» выбираются нужные группы настроек  при  определении прав и полномочий пользователя (закладка «Группы настроек в карточке пользователя», справочник «Пользователи»).  Каждый пользователь может использовать настройки только определенных групп и сохранять в эти группы собственные настройки.

Пользователь с правами «Администратор настроек пользователей» по умолчанию имеет доступ ко всем группам и может настраивать эту закладку для других пользователей.
