Справочник «Обмены данными»

 

Справочник предназначен для ввода и хранения настроек обмена данными с фронт-офисными системами.

Вызывается из меню «Сервис – Обмен данными – Обмены данными XML».

 

Настройки обмена заполняются в диалоговой форме.

В секции «Основные» указываются сведения о конфигурации, с которой производится обмен:

  • Наименование – при выборе любого из видов обмена (см. ниже), кроме «Произвольный», наименование заполняется автоматически по шаблону, например: ТHO10 --> TFO30. После этого наименование может быть отредактировано вручную. Справа указывается дата и время создания настройки;
  • Вид обмена – выбирается из выпадающего списка:
      • Обмен c «Front-Office v.3»
      • Обмен c «Front-Office v.2»
      • Произвольный.

При выборе видов обмена «Обмен c «Front-Office v.3»» или  «Обмен c «Front-Office v.2»», производится подключение настроенных правил обмена, хранящихся в конфигурации. В этом случае, два следующих реквизита заполняются автоматически, и справа от них указываются номера релизов конфигураций, для которых они предназначены.

Вид обмена «Произвольный» позволяет использовать произвольные правила обмена, загружаемые из внешнего файла в формате XML. Для этого, при выборе варианта справа становится доступна кнопка «Открыть файл», с помощью которой необходимо выбрать имеющийся файл правил.

Кнопка «Сохранить файл» позволяет сохранить правила обмена, встроенные в систему, в виде внешнего файла в  формате XML.

 

  • Конфигурация источник – указывается бэк-офисная система, из которой производится выгрузка справочников «Номенклатура», «Контрагенты», «Сотрудники» (и ряд вспомогательных), и документов «План-меню» во фронт-офисную систему.
  • Конфигурация приемник –  указывается фронт-офисная конфигурация, из которой производится загрузка данных о продажах.

 

В секции «Параметры» указываются адреса файлов данных выгрузки и загрузки (пути и имена файлов), а также  файл-флаг (служебный файл, играющий роль сигнала для фронт-офисной системы, на выполнение ею процедур импорта-экспорта данных).

 

Кнопка «Выгрузить данные» запускает универсальную обработку выгрузки данных, в соответствии с указанными в диалоговой форме правилами и именем файла данных выгрузки. Ход выгрузки отображается в окне служебных сообщений системы.  

В случае необходимости (например, если требуется задание дополнительных параметров обмена), кнопкой «Настройка» справа от кнопки «Выгрузить данные»  может быть открыто окно обработки «Универсальный обмен данными в формате XML», в которой производятся дополнительные настройки. В  таком случае, после произведенных настроек, запуск обмена данными следует производить непосредственно из формы обработки «Универсальный обмен данными в формате XML».

 

Кнопка «Загрузить данные» запускает загрузку данных о продажах из файла, указанного в поле «Файл данных загрузки». Файл данных должен быть предварительно сформирован фронт-офисной системой.
