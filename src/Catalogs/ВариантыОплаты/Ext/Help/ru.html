Справочник «Варианты оплаты».

Справочник «Варианты оплаты» предназначен для ведения списка  вариантов оплаты, используемых при оплате заказов.

Справочник вызывается из меню «Справочники - Варианты оплаты».

Данный справочник не имеет диалоговой формы, а заполняется и редактируется непосредственно в форме списка.

Поле «Тип» заполняется выбором варианта из выпадающего списка, содержащего 4 возможных значения типов оплаты:

·          Наличными,

·          Банк.картой,

·          Неплательщик,

·          Безнал.

Поле «Наименование» заполняется автоматически, но может быть отредактировано вручную.

В поле «№ в ККМ» указывается номер данного типа оплаты в фискальном регистраторе (ККМ), который может быть либо не указан, либо принимать значения от 1 до 4. Типу оплаты «Наличными» всегда должно соответствовать значение «№ в ККМ», равное 1, при типе оплаты «Неплательщик» данное поле должно оставаться незаполненным. Остальные варианты заполняются произвольно.
