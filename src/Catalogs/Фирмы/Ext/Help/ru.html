В справочник «Фирмы» необходимо занести информацию обо всех собственных юридических лицах для ведения раздельного учета по ним в одной информационной базе и для оформления документов от лица различных фирм, входящих в состав компании.

            С точки зрения финансового учета каждая фирма – это отдельный объект учета. Для каждой фирмы можно иметь свою отчетность.

Справочник предназначен для ведения списка таких фирм и хранения информации, необходимой для оформления первичных документов, ведения  учета и составления отчетности.

Справочник позволяет вести неограниченный список фирм. Информация о новой фирме и редактирование уже введенных данных осуществляется в режиме диалога. Диалог организован в виде картотеки: данные о фирме, которые вводятся в этом диалоге, распределены по нескольким закладкам. Для доступа к элементам какой-либо группы необходимо щелчком мыши открыть соответствующую закладку.

Справочник «Фирмы» целесообразно заполнить заранее, до ввода документов. Если планируется вести учет нескольких организаций, то следует ввести информацию хотя бы об одной из них. Сведения об остальных ваших организациях можно добавлять в справочник «Фирмы» по мере необходимости.

Для каждой фирмы, введенной в справочник «Фирмы», необходимо ввести хотя бы один расчетный счет. Этот счет будет использоваться в тех случаях, когда в печатных формах документов требуется указания банковских реквизитов фирмы. Некоторые реквизиты фирмы являются периодическими. Это полное наименование, руководитель, главный бухгалтер и кассир (фамилии и инициалы), почтовый и юридический адреса, контактные телефоны.

         Ввод информации о фирме

Для ввода информации о новой фирме необходимо открыть форму для ввода нового элемента справочника, нажав кнопку   Добавить(Ins) на панели инструментов. Для редактирования ранее введенной информации нужно дважды щелкнуть мышью на соответствующем элементе справочника.

Форма элемента справочника имеет пять закладок — «Основные», «Ответственные лица», «Счета», «Алкоголь» и «Учетная политика».

Сначала вводится информация о самой фирме (закладка «Основные»), а затем информация об ответственных лицах фирмы, расчетных счетах и об учетной политике фирмы. Закладка «Алкоголь» заполняется в том случае, если фирма осуществляет торговлю алкоголем.

         Закладка «Основные»

Наименование - Этот реквизит присутствует на всех закладках формы и является обязательным для заполнения.  Наименование фирмы вводится так, как оно будет выглядеть в различных списках, например, при выборе фирмы в процессе выписки первичных документов. Наименование фирмы рекомендуется заполнить без указания организационно-правовой формы и прочей дополнительной информации, чтобы было удобно его использовать при быстром поиске нужной фирмы в справочнике.

Префикс - Данный реквизит необходим для того, чтобы в финансовом учете вести отдельную нумерацию по документам различных фирм.

Контрагент – контрагент, соответствующий данной фирме. Выбирается из справочника «Контрагенты». Используется при оформлении перемещений товара между собственными фирмами торговой организации. Более подробно об этом см. в справочной информации к документу "Перемещение товаров" в подразделе «Перемещение товара между фирмами».      

Полное наименование - Наименование фирмы так, как оно должно выглядеть в первичных документах: платежных поручениях, кассовых ордерах, счетах-фактурах и других.

ИНН - идентификационный номер налогоплательщика (фирмы).

КПП – код постановки на учет в налоговые органы.

ОКПО - код юридического лица (предприятия-поставщика) по Общероссийскому классификатору предприятий и организаций.

Сайт - В данном поле указывается адрес (URL) web-сайта фирмы.

Почта - Вносится адрес электронной почты фирмы.

Значения остальных реквизитов заполняются в соответствии с их названием.

Закладка «Ответственные лица»

Руководитель, Главный бухгалтер, Кассир - В соответствующие поля формы следует внести информацию по ответственным сотрудникам фирмы из справочника «Контрагенты», заполнение этих полей необходимо для расшифровки подписей в первичных документах.

Реквизиты «Руководитель», «Главный бухгалтер» и «Кассир» являются периодическими. История периодических реквизитов справочника «Фирмы» доступна по ссылкам «История», расположенным правее полей для  ввода информации.

Закладка «Счета»

Предполагается, что фирма может иметь несколько (неограниченное количество) расчетных (банковских) счетов для осуществления денежных расчетов со своими контрагентами. Для хранения информации о расчетных счетах в конфигурации существует справочник «Банковские счета», подчиненный справочнику «Фирмы», и доступный из карточки по кнопке «Перейти».

Из всего списка банковских счетов  фирмы, один может использоваться как основной и подставляться в документы по умолчанию. Выбор основного счета осуществляется по кнопке «Основной», после чего в списке он выделяется жирным шрифтом.

Ввод и редактирование информации о банковских счетах фирмы на закладке «Счета» справочника «Фирмы»  подробно изложено в описании справочника «Банковские счета».

 

Закладка «Алкоголь»

Если фирма осуществляет торговлю алкоголем, то на этой закладке  необходимо указать сведения о лицензии, дающей право заниматься подобной деятельностью. Следует указать номер лицензии, кем она выдана и сроки ее действия.

 

         Закладка «Учетная политика»

Большинство торговых точек являются складами -  плательщиками НДС и на них сумма включает НДС. Однако, в установленном законом порядке, тот или иной склад может быть переведен в режим учета, при котором он не будет являться плательщиком НДС. Точно также, при выполнении оговоренного в законодательстве порядка, возможен и обратный переход - в режим плательщика НДС.

По умолчанию считается, что все склады являются плательщиками НДС.  

Для отражения изменений в учетной политике фирмы в отношении какого-либо склада, по кнопке «Добавить» (Ins), вызывается новая строка, на которой, путем выставления флажков, определяется, является или нет данный склад плательщиком НДС, и выставляется дата, с которой этот период наступил.
