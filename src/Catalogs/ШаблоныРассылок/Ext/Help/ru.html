Справочник "Шаблоны рассылок"

СМС-рассылки предназначены для оповещения клиентов торгового предприятия о новых товарах, услугах, маркетинговых акциях, поздравления с днем рождения и т.п. путем отправки СМС-сообщений.

Справочник «Шаблоны рассылок» – список шаблонов для автоматического заполнения текста сообщения в документе «Рассылка».  Справочник вызывается из меню «Документы»/«Рассылки»/«Шаблоны рассылок» или непосредственно из документа «Рассылка». Шаблоны создаются пользователями вручную. Количество шаблонов не ограничено.

Предопределенные шаблоны удобно использовать для организации многократных идентичных рассылок.

Новый шаблон добавляется путем нажатия на кнопку «Добавить (Ins)», после чего следует указать:

Наименование – рабочее название шаблона, применяется для выбора данного шаблона из списка.

Вид рассылки – недоступно для редактирования, заполнено по умолчанию. Имеет значение «СМС».

Далее следует ввести текст СМС-сообщения, предназначенного для отправки клиентам. При нажатии кнопки Вставить идентификатор выбранный в поле справа идентификатор вида %Клиент%  вставляется в конце текста сообщения (потом его можно перемещать «вырезанием» и последующей «вставкой» в нужное место сообщения).

При отправке сообщения идентификатор заменяется на соответствующее значение, - вместо идентификатора %Клиент% будет подставлено значение поля «Наименование» справочника «Клиенты» (как правило, это ФИО).
