Справочник «Статьи расходов».

Справочник «Статьи расходов» содержит информацию о различных типах расходов, применяемую для учета заказов с оплатой «на неплательщика», например, питание сотрудников, представительские расходы, рекламные акции и др.

Один из элементов справочника должен быть указан в качестве «Основной статьи расходов» в константах. Это та статья расходов, которая будет автоматически подставляться в заказ при выборе типа оплаты «На неплательщика», если в карточке данного клиента-неплательщика не отмечена ни одна статья (см. описание справочника «Клиенты»).

Другие элементы справочника доступны для выбора при оплате заказа способом «На неплательщика» в соответствии со списком статей расходов, указанных в карточке клиента.

В форме заполняются реквизиты:

·          Наименование – вводится произвольное наименование;

·          Код – присваивается автоматически;

·          Уровень доступа - указывает уровень доступа к статьям расходов,  выбирается из выпадающего списка.

Выбирается один из трех вариантов:

 

·       «Свободный» - статья расходов доступна для использования сотрудником с любым набором прав;

·       «Регистрация сотрудника» - при оформлении заказа на данную статью потребуется авторизация (ввод кода) сотрудника, которому дано право на подтверждение статьи расходов

·       «Регистрация клиента» - при оформлении заказа на данную статью потребуется авторизация самого неплательщика.

·          Допустимые категории - список, в котором флажками отмечаются те категории блюд и товаров, оплата которых возможна с указанием данной статьи расхода. Те категории, которые не включены в указанный список, не могут быть закрыты оплатой «На неплательщика».
