Справочник "Сотрудники".

Все сотрудники ресторана, которые работают с системой с интерфейсом «РабочееМесто», должны быть занесены в список персонала. Для этого предназначен справочник «Сотрудники» (вызов из меню «Справочники» или кнопкой  на панели инструментов).

Справочник «Сотрудники» иерархический и может иметь до трех уровней вложенности групп, что позволяет объединять сотрудников по различным признакам (уровню доступа, сменам работы и т.д.).

Информация о каждом сотруднике заполняется в форме соответствующего элемента справочника.

В форме заполняются следующие реквизиты:

·       ФИО – полное наименование (обычно - фамилия, имя, отчество сотрудника);

·       Сокращенно -  сокращенное (условное) наименование сотрудника, отображаемое после его авторизации в интерфейсе «Рабочее место»;

·       Табельный номер – реквизит, предназначенный для обеспечения однозначного соответствия (синхронизации) элементов списка сотрудников системы «Трактиръ: Front-Office ПРЕМИУМ» со справочником сотрудников товароучетной программы, с которой осуществляется обмен данными;

·       Идентификация – в этой группе указываются параметры, по которым возможно идентифицировать текущего сотрудника. Для этих целей может использоваться либо карта доступа (выбирается из справочника «Карты доступа») либо «Код доступа» (произвольный шестиразрядный цифровой код). Карта выбирается из справочника «Карты доступа». 

Если есть необходимость идентифицировать сотрудников в заказе в качестве клиентов по одной и той же карте доступа (например, для служебного питания) можно выбрать клиента, соответствующего данному сотруднику,  в реквизите «Клиент» (путем выбора из справочника «Клиенты»);

·       Права доступа  - список наборов прав данного сотрудника для каждого из мест реализации, выбирается из справочника «Права доступа». Служит для определения основного набора прав сотрудника для всех мест реализации;

·       Флаг «является бригадиром» - при установленном флажке сотрудник будет иметь доступ (в соответствии с назначенным ему набором прав) не только к своим заказам, но и к заказам своей «бригады», состав которой определяется на закладке «Состав бригады».

При установленном флаге «является бригадиром», в форме элемента справочника появляются две закладки: «Основная» (с реквизитами, описанными выше), и «Бригада».

На закладке «Бригада» указывается состав бригады сотрудника (если он является бригадиром).

В ней отображается весь доступный список сотрудников, в котором необходимо отметить флажками тех их них, кто принадлежит к бригаде.
