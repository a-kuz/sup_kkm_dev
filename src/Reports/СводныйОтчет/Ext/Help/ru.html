ОТЧЕТ "СВОДНЫЙ ОТЧЕТ"

 

При открытии отчет формируется (по умолчанию) за текущую или последнюю смену по основному месту реализации. С помощью настроек можно выбрать любой иной интересующий период формирования, и по любому месту реализации, а также сохранить эту настройку для последующего использования

 

Показатели отчета
(все показатели берутся за смену или за период, для выбранного места реализации):

 

Текущие показатели

Открыто заказов

Количество открытых заказов на данный момент

           На сумму

Сумма открытых заказов

Свободно посадочных мест

Количество посадочных мест, на которые на данный момент не оформлены заказы

Текущая загрузка зала

Количество занятых мест на данных момент, в % от всего количества посадочных мест

Свободных официантов

Количество незанятых официантов из открытой смены, которые могут принимать заказы

Текущая занятость сотрудников

Количество занятых официантов, в % от всего количества

Финансовые показатели

Выручка

Итоговая сумма выручки по всем протоколам расчетов за смену

Скидки\Наценки

Сумма по скидкам\наценкам на товар и общих скидок\наценок.

Возвраты

Сумма возвратов, оформленных за смену

Доходность

Сумма реализации

Сумма с учетом всех скидок\наценок

Себестоимость

Сумма себестоимости по всем открытым заказам за смену

Прибыль от реализации 

Сумма реализации минус себестоимость

                     Процент

Рассчитывается по формуле:

(Сумма реализации / себестоимость)*100%-100%

Средние показатели

Средняя выручка на чек

Выручка / количество оплаченных за смену

Средняя выручка на гостя

Выручка / количество гостей по всем оплаченным заказам  за смену

Среднее время обслуживания

Общая длительность всех заказов за смену * 100% / количество открытых заказов за смену

Средняя загрузка зала  (в %)

 

Общая длительность всех заказов за смену * 100%  / (количество посадочных мест данного МЕСТА РЕАЛИЗАЦИИ * длительность всей смены)

Средняя занятость сотрудников

Общая длительность всех заказов * 100% / (Количество всех сотрудников, имеющих право принимать заказы в смене * длительность смены)

Количественные показатели

Всего открыто за период  

Количество начатых (открытых) заказов

Всего оплачено за период

Количество оплаченных заказов за смену, согласно Протоколам расчетов

Всего удалено за период  

Количество удаленных заказов за смену

Количество посетителей

Количество посетителей за смену, согласно открытым заказам

Рейтинг

Самые популярные блюда                     

Три блюда, которых больше всего заказывали за смену (по количеству)

Самые активные официанты                             

Три официанта, которые приняли больше всего заказов за смену

Самые популярные места

Три посадочных места, на которые было принято больше всего заказов за смену
