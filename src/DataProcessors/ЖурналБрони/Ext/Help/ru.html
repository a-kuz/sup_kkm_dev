Обработка "Журнал брони".

Обработка "Журнал брони" используется в интерфейсе рабочего места, для того чтобы просмотреть список существующих броней, а также для ввода новых броней.

Вызывается из "Режим администратора", "Бронирование".

В журнале отображается ранее введенная бронь, попадающая в интервал действия графика бронирования, с указанием даты, времени, номера брони, клиента и посадочного места.

Ранее введенная бронь открывается двойным щелчком мыши по соответствующей строке, либо нажатием кнопки «Выбрать», когда курсор указывает на строку с бронью.

Для ввода новой брони следует нажать кнопку «Новый».  Откроется окно «Новая бронь», как описано выше, в котором можно изменить дату брони, время начала и окончания брони, клиент (выбирается из справочника «Клиенты» или вводится вручную), и посадочное место.

Кнопка «Поиск» предназначена для поиска и открытия ранее введенной брони по номеру.  При нажатии кнопки открывается диалог поиска.

Если бронь с указанным номером существует, то после ввода номера и нажатии кнопки «ОК», откроется окно с ранее введенной бронью.

Кнопка «Удалить» позволяет удалить ранее введенную бронь.

Для ввода новой брони следует нажать кнопку «Новый».
