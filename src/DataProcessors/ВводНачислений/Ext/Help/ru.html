Обработка "Ввод начислений".

Обработка «Начисления на карту» предназначена для оформления денежных расчетов с владельцами платежных карт. Начисление на карту означает увеличение остатка по карте клиента, который может быть использован клиентом при оформлении оплаты заказов.

Удержание с  карты означает уменьшение остатка по карте клиента, который может быть использован клиентом при оформлении оплаты заказов.

Вызывается из обработки "Режим администратора".

В обработки присутствуют следующие кнопки: кнопка выбора варианта оплаты, кнопка выбора ккм и кнопка выбора клиента.
