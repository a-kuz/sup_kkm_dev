Обработка "Обслуживание Windows-принтеров".

Служебная обработка, предназначенная для для настройки параметров и обеспечения взаимодействия конфигурации с Windows-принтерами.

Вызывается из справочника торговое оборудование.

В форме настраиваются следующие параметры: ширина области печати в мм, количество символов в строке, размер основного шрифта, размер двойного шрифта, имя принтера и опция предварительного просмотра для тестирования.
