Обработка "Разделение заказа".

Обработка используется для переноса всего заказа или выборочных блюд в новый заказ.

Чтобы разделить заказ и перенести блюда выборочно, выполните следующую последовательность действий:

  •     откройте список текущих заказов;
  •    найдите тот заказ, который вы хотите разделить и откройте его;
  •    нажмите кнопку «Разделение заказа»;

Заказ откроется для переноса блюд, в окне «Разделение заказа».

  • Установите курсор на строку того блюда, которое необходимо перенести. Возможно редактировать либо переносимое количество блюда (столбец «Перенести») либо его остаток в текущем заказе (столбец «Остаток»). Если блюдо не выбрано, либо переносимое количество превышает имеющееся в заказе, то будут выведены соответствующие предупреждения;   
  • Двойным щелчком мыши или кнопкой «ОК» откройте окно ввода количества,введите требуемое количество блюд;
  •  повторите операцию для всех блюд заказа, которые вы хотите перенести, при этом, по всем позициям, выбранным для переноса, в графе «Перенести» будет указано заданное количество;

Для того, чтобы при объединении заказов перенести все блюда из одного заказа в другой (не выбирая каждое блюдо и не указывая их количество), нажмите в окне «Разделение заказа» кнопку «Перенести ВСЕ». При этом, для всех блюд, имеющихся в заказе, количество будет автоматически перенесено из столбца «Остаток» в столбец «Перенести».

  • Для выбора заказа  нажмите кнопку «Выбрать заказ», для переноса позиций в новый заказ, нажмите кнопку "В новый заказ";

 

После переноса,  в исходном заказе в графе «Кол.» будут указаны оставшиеся количества по каждой позиции, перенесенные количества будут указаны в графе «Удалено», а перенесенные полностью - будут помечены в окне заказа как зачеркнутые.
