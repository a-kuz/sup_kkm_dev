Обработка "Обслуживание принтеров Атол".

Служебная обработка, предназначенная для для настройки параметров и обеспечения взаимодействия конфигурации с принтерами через драйвер производства компании "Атол".

Вызывается из справочника торговое оборудование.

Обработка содержит 4 закладки:

Закладка "Подключение".

Заполняются следующие поля: имя компьютера, к которому непосредственно подключен принтер, порт подключения к которому подключен принтер, если принтер является сетевым, то указываются его IP адрес и порт, для некоторых моделей указывается осуществлять ли контроль. Также доступна функция монопольного доступа.

Закладка "Настройки".

На этой закладке производятся настройки для печати.

Установленный признак "Проверка связи" осуществляет контроль выполнения печати, признак "Фоновая печать" позволяет отправлять задания на несколько принтеров одновременно.

Далее указываются кодировка и набор символов для печати из выпадающих списков. Указывается межстрочных интервал в мм., способ отрезки бумаги и количество пустых строк перед отрезкой.

Если для подачи сигнала подключено внешнее устройство, то необходимо установить соответствующий признак.

Закладка "Подкладной документ".

Для моделей принтеров поддерживающих функция подкладной печати, доступна закладка "Подкладной документ".

Здесь настраиваются 3 параметра: количество строк, которые будут выводится на один лист, количество строк для отступа сверху, и направление выброса бумаги после печати.

Закладка "Тест".

Для тестирования работы принтера можно вывести строку, картинку и сигнал.

для каждого из них есть свои настройки, назначение которых соответствует их названию.
