Обработка "Обслуживание принтеров СофтБаланс".

Служебная обработка, предназначенная для для настройки параметров и обеспечения взаимодействия конфигурации с принтерами чеков.

Вызывается из справочника торговое оборудование.

Обработка содержит 4 закладки:

Закладка "Подключение".

Если принтер используется удаленно, то необходимо поставить признак "Удаленное подключение" и указать IP адрес сервера печати и порт.

Для принтеров с COM и LPT интерфейсами необходимо указать порт подключения и скорость, для LAN IP адрес принтера и порт.

Кнопка "Запрос состояния" служит для проверки связи с оборудованием.

Закладка "Настройки".

Для контроля выполнения печати необходимо установить признак "Проверка связи".

Также указывается кодировка, набор символов для печати, количество пустых строк перед отрезкой и если используется для подачи сигнала внешнее устройство, то соответствующий признак.

Закладка "Подкладной документ".

Для моделей поддерживающий функцию подкладной печати доступна закладка "Подкладной документ". На ней указывается 3 параметра: Количество строк, которые будут размещаться на листе, количество строк для отступа сверху и направление выброса бумаги после печати.

Закладка "Тест".

На данной закладке можно настроить различные параметры для тестовой печати строк, картинок и сигнала. Значения всех параметров соответствуют их названию в форме.
