Обработка "Выбор смен".

Обработка предназначена для выбора смен при формировании отчетов, когда отчеты формируются по сменам.

Необходимо указать место реализации, по которому будут выбраны смены.

В табличной форме указываются непосредственно смены, за которые необходимо сформировать отчет.

Вызывается из отчетов, предусматривающих формирование данных по сменам.
