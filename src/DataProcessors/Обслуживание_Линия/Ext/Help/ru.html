Обработка "Обслуживание коммутаторов электропитания".

Служебная обработка, предназначенная для для настройки параметров и обеспечения взаимодействия конфигурации с коммутаторами электропитания.

Вызывается из справочника торговое оборудование.

Содержит 2 закладки:

Закладка "Подключение".

Если коммутатор подключается по сети, то необходимо установить признак "Удаленное подключение" и указать IP адрес сервера и порт.

В поле порт, необходимо указать порт  компьютера, к которому подключен коммутатор.

По кнопке "Запрос состояния" можно осуществить проверку связи с оборудованием.

Закладка "Управление каналами".

В поле номер канала можно выбрать канал. Для него будут доступна следующие действия: включить, отключить и проверить состояние.

Соответствующая информация будет отображена справа.
