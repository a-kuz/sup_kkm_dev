Обработка "Обслуживание фискальных регистраторов Штрих".

Служебная обработка, предназначенная для для настройки параметров и обеспечения взаимодействия конфигурации с фискальными регистраторами через драйвер разработки компании "Штрих".

Вызывается из справочника торговое оборудование.

Обработка содержит 5 закладок:

Закладка "Подключение".

На этой закладке необходимо указать имя компьютера, если работа с фискальным регистратором происходит по сети. Порт подключения и скорость обмена данными.

Далее необходимо указать таймаут и пароль доступа к ККМ, и если нет необходимости вести учет продаж в разрезе кассиров, то признак того, что используется один пароль для всех кассиров.

По кнопке "Параметры ККМ" открывается тест драйвера, подробнее о настройках которого можно прочитать в документации к драйверу.

Закладка "Список кассиров".

Если в предполагается вести учет продаж в фискальном регистраторе в разрезе кассиров, то необходимо заполнить список кассиров, путем подбора их справочника "Сотрудники". 

Кнопка "Прочитать" служит для чтения списка из фискального регистратора, кнопка "Загрузить" служит для загрузка в ФР заполненного списка кассиров.

Закладка "Время".

Для корректной работы связки ККМ-конфигурация необходимо чтобы время между ними отличалось не более чем на 5 минут.

Для синхронизации служит одноименная кнопка. Для обновления данных в форме кнопка "Обновить".

Закладка "Шапка фискального документа".

Можно настроить шапку фискально документа, для этого необходимо поставить соответствующий признак и ввести значение шапки.

Закладка "Подкладной документ".

Для моделей ФР, которые поддерживают функцию подкладного документа, доступна данная закладка.

Для того, чтобы печать нефискальных документов происходила на станции подкладной печати, необходимо установить соответствующий признак.

также указывается количество строк на листе, количество символов в строке, отступ сверху.
