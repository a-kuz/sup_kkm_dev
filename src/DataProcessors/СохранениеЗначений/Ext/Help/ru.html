Сохранение значений

        Обработка предназначена для сохранения значений конфигурации и вызывается только из других объектов конфигурации.

        Для сохранения значений необходимо указать группу настройки (справочник "Группы настроек") и ее наименование.

        Если предполагается, что настройка будет перезаписана поверх уже существующей настройки, то производится двойной клик мышью по настройке поверх которой будет производиться запись.
