Обработка "Обслуживание считывающих устройств Атол".

Служебная обработка, предназначенная для для настройки параметров и обеспечения взаимодействия конфигурации со считывающими устройствами через драйвер разработки компании "Атол".

Вызывается из справочника торговое оборудование.

Поддерживается работа с тремя видами считывающих устройств: сканеры штрих-кодов, проксимити-считыватели и ридеры магнитных карт.

Обработка содержит 2 закладки:

Закладка "Параметры".

Указывается номер логического устройства, по кнопке "Свойства" открывается тест драйвера, где необходимо указать все необходимые параметры. Подробнее см. документацию к драйверу.

Закладка "Тест".

Для тестирования работы устройства необходимо поставить признак того, что устройство включено. Далее необходимо считать информацию, которая отобразится в поле "Результат".
