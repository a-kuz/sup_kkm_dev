Выбор значений для отбора

В данном окне можно указать те значения, которые будут участвовать в отборе. В списке Доступные значения представлены возможные значения отбора, а в списке Выбранные значения - установленные значения отбора.

Для выбора нужного значения отбора необходимо выбрать это значения в списке Доступные значения и нажать кнопку Добавить >. Для выбора всех доступных значений следует нажать кнопку Добавить все >>.

Если какое-либо значение следует исключить из списка Выбранные значения (т.е. исключить значение из отбора), то следует нажать кнопку < Удалить, предварительно выбрав нужное значение. Для очистки списка выбранных значений следует нажать кнопку << Удалить все.
