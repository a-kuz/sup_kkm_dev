Указание отбора по разделителям

Для задания отбора по разделителям в форме отбора журнала регистрации в поле "Разделение данных сеанса" следует нажать кнопку выбора.

В форме показывается список разделителей, по которым в журнале регистрации зарегистрированы события.

В списке следует установить флажок для того разделителя, по которому требуется выполнить отбор и в правой колонке указать значение разделителя. Значение выбирается в специальной форме, открываемой кнопкой выбора данного поля. В этой форме требуется установить флажки для тех значений разделителя, по которым будет выполнен отбор, или выбрать значение "<Не задано>". В последнем случае будет выполнен отбор событий, для которых не установлено значение разделителей, т.е. сделанных в неразделенном режиме.
