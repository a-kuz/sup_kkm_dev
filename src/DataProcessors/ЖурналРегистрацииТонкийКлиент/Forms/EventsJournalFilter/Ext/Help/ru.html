Установка отбора журнала регистрации

С помощью настроек отбора можно установить фильтры событий.

Для временного ограничения отбора событий в поле Интервал отображения указывается период, за который следует отобрать события.  

Поле отбора Разделение данных сеанса не отображается, если нет разделителей, которые допустимо использовать в пользовательском отборе, т.е. таких, которые в данном сеансе не используются (не установлены, выключены). В поле задаются значения разделителей, по которым требуется выполнить отбор в журнале регистрации.

В группе Основные свойства задается отбор по пользователям, событиям, компьютерам, приложениям, комментариям, а также по важности события. Ввод значения выполняется нажатием кнопки выбора (...) рядом с полем.

В группе Данные задается отбор по метаданным, данным и представлениям этих данных.

В группе Прочие задается отбор по транзакции, сеансам, рабочим серверам, по основным и вспомогательным портам.

Поле Сеансы позволяет вручную ввести список сеансов. Номера сеансов отделяются друг от друга запятыми.

Транзакция определяется видом (в списке Статус транзакции установите флажки для тех видов, которые должны попасть в отбор). Также можно указать номер транзакции.
